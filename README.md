# 🌙 Moon Sighting Night - Interactive Hunt

Ek khoobsurat aur interactive web project jo **Chand Raat** ki munasbat se banaya gaya hai. Ismein ek "Spotlight" effect hai jahan user ko andhere mein chhupa hua Chand dhoondna hota hai.

---

## 🚀 LIVE DEMO
Aap is project ka interactive version yahan live dekh sakte hain:
### 👉 [Click Here to Play Moon Hunt](https://hs5042708-web.github.io/Moon-Sighting-Night-/)

---

## ✨ Key Features
- **Spotlight Effect:** Mouse cursor ek flashlight ki tarah kaam karta hai.
- **Hidden Moon Logic:** Chand ko har baar refresh karne par random position par hide kiya jata hai.
- **Neon Aesthetic:** Luxury dark blue theme aur gold neon glow effects.
- **Minimalist Code:** Clean HTML, CSS, aur Vanilla JavaScript ka istemal.

## 🛠️ Technologies Used
- **HTML5:** Structure.
- **CSS3:** Animations, Radial Gradients, aur Spotlight layers.
- **JavaScript:** Mouse tracking logic aur event listeners.
