# 🌙 Moon Sighting Night - Interactive Hunt

A beautiful and interactive web application designed to celebrate the festive occasion of **Chand Raat**. This project features a unique "Spotlight" navigation system where users must explore the dark night sky to discover the hidden crescent moon.

## ✨ Key Features
- **Spotlight Navigation:** The mouse cursor acts as a functional flashlight, revealing hidden elements in the dark.
- **Dynamic Hidden Logic:** The moon is programmatically hidden at a random position on every page refresh for a new experience.
- **Luxury Neon Aesthetic:** High-contrast dark blue theme with premium gold neon glow effects.
- **Optimized & Minimalist:** Lightweight build using clean HTML, modern CSS, and Vanilla JavaScript.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Advanced Radial Gradients, custom spotlight layers, and keyframe animations.
- **JavaScript (ES6):** Real-time mouse tracking logic and interactive event listeners.
